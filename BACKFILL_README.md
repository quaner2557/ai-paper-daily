# 回刷脚本使用说明

## 快速开始

回刷 2025 年 1 月整月数据：

```bash
cd ~/.openclaw/workspace/ai-paper-daily

# 标准模式（每个日期间隔 2 秒，避免 API 限流）
python backfill.py --start 20250101 --end 20250131

# 快速模式（无延迟，适合测试）
python backfill.py --start 20250101 --end 20250131 --no-delay

# 自定义延迟（比如 5 秒）
python backfill.py --start 20250101 --end 20250131 --delay 5
```

## 功能特性

✅ **自动去重**：扫描 output 目录，跳过已存在的日期  
✅ **幂等处理**：同一日期多次运行不会重复  
✅ **限流保护**：可配置延迟，避免触发 arXiv API 限流  
✅ **完整输出**：同时生成 JSON/Markdown/HTML 三种格式  
✅ **错误处理**：失败日期会记录日志，可单独重试

## 注意事项

1. **API 限流**：arXiv API 有速率限制，建议使用默认 2 秒延迟
2. **LLM 成本**：每篇论文需要调用 LLM 评分，注意 token 消耗
3. **网络稳定**：确保网络通畅，避免中途失败
4. **失败重试**：如果某天失败，可以单独重试该日期

## 单独重试某天

```bash
# 只处理 20250115 这一天
python backfill.py --start 20250115 --end 20250115 --delay 2
```

## 查看进度

```bash
# 查看已处理的日期
ls -la output/*.json | wc -l

# 查看具体某个日期的文件
ls -la output/202501*.json
```

## 环境变量

确保 `.env` 文件配置正确：

```bash
# LLM 配置
LLM_API_KEY=your_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_PRERANK_MODEL=qwen3.5-flash
LLM_FINERANK_MODEL=qwen3.5-plus

# arXiv 配置（可选）
ARXIV_CATEGORIES=cs.IR,cs.LG,cs.AI,cs.CL
MAX_PAPERS_FETCH=400
MAX_PAPERS_OUTPUT=50
MIN_RELEVANCE_SCORE=4
```
